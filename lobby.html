FAQ - Perguntas Frequentes

Como posso adicionar minhas plantas?

Para adicionar suas plantas, clique no botão "Minhas Plantas" no menu principal e siga as instruções para cadastrar suas plantas.

Como usar o mapa?

O mapa permite visualizar a localização das suas plantas e arquitetos próximos. Clique no botão "Mapa" para acessar essa funcionalidade.

Como iniciar um chat com um arquiteto?

Clique no botão "Chat com Arquiteto" para iniciar uma conversa com um arquiteto disponível para ajudar você com seu projeto.

Como editar meu perfil?

Para editar seu perfil, clique no botão "Perfil" e acesse as configurações para atualizar suas informações pessoais.
